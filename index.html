Press Enter to start the Tutorial Mode.
Your time will be paused when you're still in this mode.

When the countdown starts,
you can see how much time you have left here.

On every location in the scene, there will be some objects that you can pick up.
Use Left & Right buttons to select an object.

Press Down to observe the object in CloseUp Mode.

Press Enter to store the object in Inventory (exit CloseUp Mode first).
This action cannot be undone!
To open Inventory, press Up.

When you're ready, press PageUp to enter the scene and start the countdown.
Once the scene starts, you can use PageUp and PageDown to move between different locations in the scene.
